As our website idea becomes clearer, I started to think more about whether the website is actually suitable for the target users. Since our website is for middle-aged and older adults, it is not enough to only make the functions work. The website also needs to be easy to understand and comfortable to use.

One important thing is accessibility. The text should not be too small. The colour contrast should be clear. Buttons should be easy to find. The website should not have too many things on one page. I think this is important because some users may not be very confident with technology. If the website feels complicated, they may stop using it.

Another important thing is the post creation process. The form should be simple. Users should not need to fill in too many fields just to make one post. For example, they may only need to write the content, choose a category, and submit. If we add too many options, the function may become harder to use.

Privacy is also something we need to think about. A social website means users may share personal information, photos, or daily routines. Even though this is only a prototype, we should still think about what information is public and what information should not be collected. We should not ask users for unnecessary personal details.

Safety is another issue. If users can post and comment, there may be inappropriate content. We may not be able to build a full moderation system, but we can still include a simple report button. This shows that we have thought about the possible risks of social interaction.

For the next steps, our group needs to finish the wireframes and make sure the main pages are clear. We also need to plan the data structure, such as users, posts, comments, tips, and coupons. After that, we can focus more on building the actual prototype.

My main reflection is that a good prototype does not need to have many complicated functions. For our project, it is more important that the website feels simple, useful, and friendly. If users can easily post, read, comment, and find useful tips or coupons, then the main purpose of the website is already clear.
