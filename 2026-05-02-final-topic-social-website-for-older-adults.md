This week, our group finally decided our topic. We are going to build a social website for middle-aged and older adults. The main purpose of this website is to give them a simple online space where they can communicate with other people, share daily life, and get some useful information.

The most basic function of the website is social posting. Users should be able to see posts from others and create their own posts. This is the foundation of the whole website. If users cannot post or read posts, then the website is not really a social platform. So this function needs to be one of the first things we build.

The second function is interaction. Users should be able to comment on posts or like them. I think this is important because social connection is not only about posting. It is also about getting responses from other people. Even a simple comment function can make the website feel more like a community.

The third function is life tips. We discussed that middle-aged and older users may enjoy sharing useful knowledge from daily life. For example, they might share cooking tips, shopping tips, health routines, or technology help. This function makes the website more useful, not just a place to chat.

The fourth function is coupons or platform benefits. We want the website to have a section where users can see supermarket coupons or other benefits. This is not the main social function, but it can make the website more attractive. It also gives users another reason to come back to the platform.

When thinking about the design, I feel the website should be simple and clear. It should not look too crowded. The words should be easy to read, and the buttons should be easy to understand. I do not think we should design it like a very fast social media app, because that may not match the needs of our target users.

At this point, our topic is much clearer than before. The website is not only “for older people”. It is a social and practical platform. The main idea is to combine communication, daily sharing, useful tips, and benefits in one simple website.
